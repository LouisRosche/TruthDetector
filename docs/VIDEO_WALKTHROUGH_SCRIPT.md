## Production Notes

### Equipment Needed
- **Screen recorder**: OBS Studio (free), Camtasia, or ScreenFlow
- **Microphone**: USB condenser mic (Blue Yeti, Audio-Technica AT2020)
- **Webcam**: 1080p minimum (if including talking head)
- **Editing software**: DaVinci Resolve (free), Adobe Premiere, or iMovie

### Recording Tips
1. **Screen resolution**: Record at 1920x1080 (1080p)
2. **Frame rate**: 30fps minimum
3. **Audio quality**:
   - Record in quiet room
   - Use pop filter
   - Separate audio track from system sounds
4. **Cursor**: Use cursor highlighting (OBS plugin or Camtasia feature)
5. **Pacing**: Speak slowly and clearly (teachers are often multitasking)

### Editing Checklist
- [ ] Add chapter markers for YouTube (Setup, Gameplay, Dashboard, Tips)
- [ ] Include captions/subtitles (auto-generate + manual review)
- [ ] Add background music (royalty-free, low volume 10-15%)
- [ ] Zoom in on small UI elements (buttons, text fields)
- [ ] Add "Click here" callouts for clarity
- [ ] Include lower-third text for key concepts
- [ ] Export at 1080p, 30fps, H.264 codec

### Accessibility
- **Closed captions**: Generate via YouTube auto-captions, then edit for accuracy
- **Audio description**: Consider separate audio-described version for visually impaired
- **Transcript**: Provide full text transcript in video description
- **Color contrast**: Ensure on-screen text is high contrast (white text on dark background)
